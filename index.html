About Me

저는 운영 환경에서 발생하는 데이터 처리 문제를 해결하며 성장해 온 백엔드 개발자입니다. 이전 회사에서는 대기업 운영 시스템을 개발하며 대용량 조회 성능을 개선하고, 배치 처리 과정에서 발생하던 데이터 누락 문제를 보완했습니다.

현재는 전기차 화재 징후를 감지하는 열화상·가스 센서 안전관제 시스템을 개발하고 있습니다. 센서 데이터 수집부터 저장, 분석, 리포팅까지 이어지는 흐름을 다루며, 실시간 데이터가 안정적으로 처리될 수 있는 구조를 고민하고 있습니다. 필요할 때는 익숙하지 않은 영역도 직접 맡아 C++ 기반 다중 카메라 서버의 동시성 문제를 개선한 경험이 있습니다.

개인 프로젝트를 통해서는 데이터 동기화, 검색 성능 개선, 로그 추적 같은 백엔드 구조를 직접 구현하며 학습해 왔습니다. 저는 단순히 기능을 구현하는 것보다, 운영 중에도 안정적으로 동작하고 문제가 반복되지 않는 시스템을 만드는 개발자가 되고자 합니다.
