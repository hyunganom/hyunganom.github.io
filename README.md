# 이현재 GitHub Pages 이력서 & 포트폴리오

GitHub Pages 배포용 정적 웹사이트입니다.

## 포함 페이지

- `/` : 웹 이력서
- `/portfolio/` : 포트폴리오 목록
- `/portfolio/city-safe/` : CITY_SAFE 전기차 배터리 화재·과열·이상 감지
- `/portfolio/city-signal/` : CITY_SIGNAL 도시 교통신호 최적화
- `/portfolio/thermal-parking/` : 열화상 주차 감지 CV 알고리즘
- `/portfolio/kafka-event-sync/` : Kafka 실시간 이벤트 데이터 동기화 전략
- `/portfolio/kafka-es-search/` : Kafka 이벤트 동기화 기반 Elasticsearch 검색 아키텍처
- `/portfolio/elk-log-tracking/` : ELK RequestId·MDC 로그 트래킹 전략

## PDF 출력

각 상세 페이지에는 `브라우저에서 PDF 저장` 버튼이 있습니다.
또한 `assets/html-pdf/` 폴더에 HTML 기준으로 미리 출력한 PDF 파일을 포함했습니다.

## 배포 방법

압축을 푼 뒤 ZIP 파일 자체가 아니라 내부 파일 전체를 GitHub 저장소 루트에 업로드하세요.
